# Stress + Perdita di massa muscolare 

## Dati in ingresso wearable:

- Calorie
- Sonno 
- Passi
- Heart rate

## Dati non wearable (ESSENZIALI / NON ESSENZIALI) : 

- Sesso
- eTà
- LAVORO
- PESO
- ALTEZZA
- Come vai a lavoro 
- Attività fisica
- Patologie specifiche 

###  Output: 
-  Indice di stress (basso, medio, alto)
-  Notifica di sedentarietà 
-  Invio dei dati al datore di lavoro per supervisionare il benessere ( il datore si vanta che i suoi dipendenti stanno bene) 
-  Trend dell'indice ( per 3 giorni sei basso, conviene che riduci il lavoro) 
-  Resoconto settimanale, mensile e annuale. (Girono, mese più stressante)
-  Confronto fra reparti (+ con media locale/nazionale)

### Funzionalità aggiuntive: 

- Esercizi posturali
- Questionario sulla giornata (quantificabile con le faccine)
- Warning ( notifiche )


Target: 
- Lavoratori d'ufficio
- Camionisti / autisti autobus / piloti 
- Infermieri / medici 

Futturi sviluppi: 
- Mappa dello stress (luoghi e lavori)
- Classificare le aziende per stress ( anche per reparti )
